# SKILL: Intigriti Tactics
## Version: 1.0 | Domain: platform-intel

---

## STRENGTHS
- EU-based; strong on financial / e-commerce programs.
- Friendly to chained / business-logic bugs.
- Monthly top-10 researcher leaderboard with cash bonus.
- Their CTF events ("1337UP") help build rep + invite to private programs.

## TRIAGE
- Triagers respond reasonably fast (median <72h).
- Open to negotiation on severity if you cite CVSS + impact clearly.

## SCOPE QUIRKS
- Many programs allow `automated scanning is OK if rate-limited` — explicit permission.
- Always read the "Out of scope" carefully — some allow vulns in 3rd-party widgets.

## PAYOUT
- Bonus tiers for severity above expected (e.g., reflected XSS escalated to ATO can hit P1 tier).
- Quarterly bonus pools for top researchers.

## REPORTS
- Use their "Confidence" field — set to "High" for verified bugs only.
- Keep title concise; their Slack-style notification truncates.

go.intigriti.com/researcher, intigriti.com/leaderboard
